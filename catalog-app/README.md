# Groopy WhatsApp Catalog 🚀

A high-performance, mobile-first product catalog built for product importers.

## ✨ Features
- **Modern UI**: Hebrew (RTL) support with premium Assistant typography.
- **Product Gallery**: Dynamic grid with quick search by Name or SKU.
- **Category Filtering**: Easily toggle between Bottles and Lunch Boxes.
- **Smart Shopping Cart**: Real-time quantity adjustments and total price calculation.
- **WhatsApp Integration**: Generates a perfectly formatted Hebrew order message with customer details.
- **Responsive Design**: Optimized for mobile users (majority of the user base).

## 🛠️ Tech Stack
- **React**: Modern component-based architecture.
- **Tailwind CSS**: Custom utility-first styling with premium gradients.
- **Framer Motion**: Smooth micro-interactions and transitions.
- **Lucide React**: Beautiful icons for a professional look.

## 🚀 Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Build for production: `npm run build`

## 🎨 Visual Style
Inspired by the vibrant cyan and orange gradients of the Uzspace product line.
